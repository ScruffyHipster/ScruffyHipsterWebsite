Wren — Workout Recovery Engine

Current MVP: a fast, configurable round timer with presets for MMA, Boxing, and Grappling. Built natively with SwiftUI. HealthKit/AI insights planned.

See features

Built for better training days

A focused training tool: set rounds, rest, and warnings fast; run reliable sessions with discipline‑specific presets. Insights via HealthKit will come later.

Round Timer

Warm‑up → rounds → rests → cooldown, with clear countdown and large, glanceable UI.

Discipline Presets

One‑tap profiles for MMA, Boxing, and Grappling; customize rounds, rest, and warnings.

Audio Cues

Distinct start/end/warning sounds; configurable volume; designed for noisy gyms.

Session Logs

Optional lightweight logging (date, type, duration) with SwiftData for future trends.

Screenshots

Drop your assets into /assets/ and keep these filenames to wire them up instantly.

Privacy first

No Health data is used in the current MVP. Future versions will request explicit permissions for HealthKit features and keep processing on‑device where possible.
